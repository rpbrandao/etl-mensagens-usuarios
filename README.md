# ETL de Mensagens Personalizadas

## 📌 Descrição
Projeto simples de pipeline ETL em Python que lê dados de usuários, gera mensagens personalizadas e salva o resultado.

## 🚀 Tecnologias
- Python
- Pandas

## 📂 Estrutura
```
etl-mensagens-usuarios/
│
├── data/
├── src/
├── README.md
├── requirements.txt
```

## ▶️ Como rodar
```bash
pip install -r requirements.txt
python src/main.py
```

## 🧠 Aprendizados
- Conceito de ETL (Extract, Transform, Load)
- Manipulação de dados com Pandas
- Organização de projeto em Python
